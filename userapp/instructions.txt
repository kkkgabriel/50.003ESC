Installation:

1. install React
npm install -g create-react-app

2. install kendoUI
npm install --save @progress/kendo-react-conversational-ui
npm install --save @progress/kendo-react-buttons
npm install --save @progress/kendo-react-intl
npm install --save @progress/kendo-theme-default

3. install Dialogflow
npm install dialogflow

4. install express
npm install express

5.install uuid
npm install uuid

6. install axios

6.install Moesif CORS extension on your chrome

HOW TO USE:
1. cd to the src folder with test.js
initialise dialogflow server by running in cmd:
node test.js

2. run react app with npm start

IF THERE ARE ANY ISSUES, make sure server is running on localhost:3000, and react app is running on another port.
